### 🛠 Installing

In the project directory, 

1. Install dependencies

   

``` bash
   npm install
   ```

2. Run the app in the development mode

   

``` bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### 🚀 Build the app and minify for production

``` bash
npm run build
```
